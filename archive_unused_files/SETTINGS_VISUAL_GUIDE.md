## 📏 UI MEASUREMENTS

### Spacing:
- Card Padding: 16px
- Section Spacing: 24px
- Element Spacing: 12px
- Icon Size: 24px
- Profile Picture: 80px (Resident), 100px (Edit mode)

### Borders:
- Border Radius: 12px (cards), 8px (inputs)
- Border Width: 1px
- Border Color: Grey[300]

### Typography:
- Section Title: 20px, Bold
- Card Title: 18px, Bold
- Body Text: 16px, Regular
- Subtitle: 14px, Regular
- Caption: 12px, Regular
- Button Text: 16px, Bold

### Buttons:
- Height: 48px (primary), 40px (secondary)
- Border Radius: 12px (primary), 8px (secondary)
- Padding: 16px vertical, 24px horizontal

- ✅ Modal slide-in animations

## 📱 RESPONSIVE DESIGN

### Mobile Optimizations:
- ✅ Single column layout
- ✅ Full-width cards
- ✅ Touch-friendly tap targets (min 48px)
- ✅ Scrollable content
- ✅ Keyboard-aware forms
- ✅ Bottom padding for floating action buttons

### Tablet Support:
- ✅ Same layout (mobile-first)
- ✅ Better use of space with padding
- ✅ Larger profile pictures on larger screens

## 🔄 STATE MANAGEMENT

### Local State (StatefulWidget):
- Edit mode toggle
- Form field values
- Toggle switch states
- Loading indicators
- Contact list

### Persistent State (SharedPreferences):
- User profile data
- Notification settings
- Privacy settings
- Emergency contacts

### Sync State:
- Emergency contacts shared between screens
- Real-time updates via service
